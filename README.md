## ✨ Features

- **Multiple Strategy Support**: Includes various quantitative strategies like Moving Average Crossover, RSI Mean Reversion, Bollinger Bands, etc.
- **Kite Connect Integration**: Fetch historical data directly from Zerodha's Kite Connect API
- **Comprehensive Backtesting Engine**: Realistic trade simulation with slippage and brokerage calculations
- **Performance Analytics**: Detailed metrics including Sharpe ratio, max drawdown, win rate, etc.
- **Visualization**: Interactive charts for equity curves, drawdowns, and trade analysis
- **Parameter Optimization**: Grid search functionality for strategy parameter optimization
- **Risk Management**: Built-in position sizing and stop-loss mechanisms

## 🔧 Prerequisites

- Python 3.8+
- Kite Connect API credentials (API key, Access token)
- Zerodha trading account
